# Note Taking Application (Flask)

A simple Flask-based note taking application where users can add notes and view them on the same page.

## Features
- Single home route
- Add notes using a form
- Display notes as an unordered list
- Simple and clean UI

## Tech Stack
- Python
- Flask
- HTML (Jinja2)

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install flask
3. Run the app:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/

## Bugs Fixed
- Incorrect HTTP method handling
- Form submission issues
- Improper request data fetching
- Missing GET route
